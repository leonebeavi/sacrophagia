# Sacrophagia

Sistema de RPG autoral e pessoal em desenvolvimento — horror teológico num mundo pós-apocalíptico. Ao partir o átomo, a humanidade rasgou a fronteira com o Além: os antigos reatores nucleares tornaram-se sarcófagos que aprisionam deuses, e o mundo desabou em ruínas famintas. No que restou, devorar é a lei — a graça, a carne e o próprio sagrado são coisas que se comem.
